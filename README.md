# AI Training Workshop Repo

This repository contains workshop materials for teaching practical AI-assisted development workflows with VSCode and GitHub Copilot.

## Repository Structure

- `deck/`: workshop slide deck and facilitation materials.
- `demo-app/`: hands-on TypeScript/Express app for exercises.

## Deck Contents

- `deck/workshop-deck.md`: Marp slide deck.
- `deck/facilitator-guide.md`: run-of-show and facilitation notes.
- `deck/live-demo-script.md`: live demo prompts and flow.

## Demo App Contents

- `demo-app/README.md`: app setup and exercise alignment.
- `demo-app/exercises/`: exercise instructions.
- `demo-app/.github/prompts/`: reusable Copilot prompt files.
- `demo-app/.github/skills/`: Copilot skills using `skill-name/SKILL.md` convention.

## Quick Start

```bash
cd demo-app
npm install
npm test
npm run dev
```

## Workshop Focus

- Prompt quality and iteration
- Planning vs implementation workflows
- Debugging and refactoring with AI assistance
- Converting ad-hoc prompts into reusable team skills
