# AI-Assisted Development Workshop Pack

This folder contains a stronger, delivery-ready workshop deck and facilitation materials.

## Files

- `workshop-deck.md`: 90-minute core deck (with optional 2-hour expansion).
- `facilitator-guide.md`: run-of-show, speaking notes, and demo choreography.
- `live-demo-script.md`: exact commands and prompts for live delivery.

## Audience

Developers already shipping code who need practical, repeatable AI workflows with GitHub Copilot, VSCode, and terminal assistants.

## Session Outcome

Teams leave with:

1. A shared mental model: Prompt -> Skill -> Workflow.
2. A concrete repo structure for `.github/prompts` and `.github/skills`.
3. A working demo app to practice generation, testing, debugging, and refactoring.
